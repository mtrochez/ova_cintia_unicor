OVA Cálculo 1 - Unidad 1 Tema 1

Descripción General

Objeto Virtual de Aprendizaje interactivo para el curso de "Cálculo 1" de Ingeniería de Sistemas, enfocado en **Funciones y Gráficas**.

Objetivos de Aprendizaje
- Comprender y analizar el concepto de función
- Identificar dominio, rango y tipos de funciones
- Interpretar y construir representaciones gráficas
- Desarrollar habilidades para resolver problemas aplicados a Ingeniería de Sistemas

Características Principales

Simulador Gráfico Interactivo
- Editor de funciones con entrada en tiempo real
- Gráfica dinámica con ejes y escalas ajustables
- Análisis automático de dominio, rango, simetría y comportamiento
- Prueba de recta vertical animada
- Detección de extremos (máximos y mínimos)

Actividades Educativas
- 3 desafíos guiados con retroalimentación inmediata
- Sistema de progreso con barra de avance
- Problemas contextualizados para Ingeniería de Sistemas
- Aprendizaje activo y experimental

Evaluación y Recursos
- 10 preguntas interactivas alineadas con el contenido
- Retroalimentación específica para cada respuesta
- Recursos digitales (Khan Academy, Wolfram Alpha, GeoGebra)
- Bibliografía actualizada del curso

Cómo Usar

Simulador Interactivo (Recomendado)
1. Ingresa una función matemática (ej: x^2, sin(x), sqrt(x))
2. Observa la gráfica en tiempo real
3. Usa las herramientas de análisis para explorar propiedades
4. Completa los 3 desafíos guiados


OVA Completo
1. Introducción: Contexto del tema
2. Objetivos: Metas de aprendizaje
3. Contenido: Teoría detallada
4. Actividades: Simulador y ejercicios
5. Evaluación: Quiz interactivo

Funciones Disponibles en el Simulador

Funciones Básicas
- `x^2`, `x^3` - Polinomios
- `sqrt(x)` - Raíz cuadrada
- `abs(x)` - Valor absoluto
- `1/x` - Función racional

Funciones Trigonométricas
- `sin(x)` - Seno
- `cos(x)` - Coseno
- `tan(x)` - Tangente

Operaciones
- Suma: `x + 2`
- Resta: `x - 3`
- Multiplicación: `2*x`
- División: `x/2`
- Potencias: `x^2`

Desafíos Guiados

Desafío 1: Dominio y Rango
- Función: f(x) = √(9-x²)
- Objetivo: Determinar dominio y rango
- Herramienta: Simulador gráfico

Desafío 2: Simetría
- Función: f(x) = x³ - 3x
- Objetivo: Identificar si es par, impar o ninguna
- Herramienta: Análisis de simetría

Desafío 3: Aplicación
- Problema: Área de rectángulo con perímetro 20
- Objetivo: Optimización usando funciones
- Herramienta: Modelado matemático

Evaluación
Contenido del Quiz
- 10 preguntas sobre funciones y gráficas
- Dominio y rango de funciones específicas
- Simetría (par/impar)
- Prueba de recta vertical
- Comportamiento de funciones

Retroalimentación
- ✅ Correctas: Explicación detallada
- ❌ Incorrectas: Guía para la respuesta correcta
